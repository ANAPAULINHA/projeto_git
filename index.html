git init – Inicializar o repositório dentro de uma estrutura. git clone – Obter uma estrutura de protejo de um servidor remoto, EX: GITHUB para ser commitados git commit -m “mensagem de commit” - Salvar os arquivos em commit. Aclopar alterações -m especifica a mensagem de commit git push – Publicar alteração no servidor remoto git pull – Você baixas alterações do no seu repositório local, atualiza o seu repositorio local, de acordo com seu repositório remoto. git merge – Juntar duas branch. Resolução de conflito. git status – Prover informação referentes ao estado atual do nosso projeto. Se tem alterações, se tem commit... git log – Relatório de alterações
